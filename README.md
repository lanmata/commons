# Commons

## Status

**Spring Boot Version**: 3.5.8  
**Java Version**: 21  
**Spring Framework**: 6.2.3

## Overview

PRX Commons is a shared library providing common components, utilities, and constants for PRX applications built with Spring Boot 4 and Java 25.

## Migration

This project has been upgraded to Spring Boot 3.5.8 and Java 21.

## Building

```bash
# Requires Java 21
mvn clean package
```
