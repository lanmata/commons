# Commons

## Status

**Spring Boot Version**: 4.0.0  
**Java Version**: 25  
**Spring Framework**: 7.0.1  
**Spring Cloud**: 2025.1.0 (Oakwood)

## Overview

PRX Commons is a shared library providing common components, utilities, and constants for PRX applications built with Spring Boot 4 and Java 25.

## Migration

This project has been upgraded to Spring Boot 4.0.0 and Java 25. See [MIGRATION_SPRING_BOOT_4.md](MIGRATION_SPRING_BOOT_4.md) for detailed migration documentation.

## Building

```bash
# Requires Java 25
mvn clean package
```

- Spring Boot 4.0.0
